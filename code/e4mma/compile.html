Compiling E4MMA¶

Compiling the code requires the installation of Boost, GSL (versions 2.0 and later), HDF5, and O2scl. You will need to manually edit the makefile to work with your system and then compile eos_nuclei in order to generate an EOS.

Alpha versions of E4MMA are currently built around specific versions of O2scl/O2sclpy and commits from the andrew branch of E4MMA.

E4MMA Alpha 5¶

  • O2scl/py v0.930a4 (based on docker file v0.930a4_u24.04_py) and E4MMA commit b4ea4a7c. See E4MMA docker file alpha5_ju_o0930a4_u24.04.
